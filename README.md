### 2.linux下安装docker
 
 1. 使用curl命令下载shell脚本<br/>
    curl -fsSL get.docker.com -o get-docker.sh
 2. 使用sh命令执行这个脚本<br/>
    sh get-docker.sh
 3. 检查是否安装成功<br/>
    docker version
 4. 开启docker服务端<br/>
    sudo systemctl start docker
